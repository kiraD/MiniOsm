Lire le README.md /!\ Au démmarage, GoldenSun Map theme song se lance.

Pour compiler le programme faire la commande make (dans source)L'option -p dessine mieux.
ensuite pour exécuter ./projet map.osm :

Vous pouvez vous déplacer et zoomer sur la map :
-Les touches /!\ :QSDZ pour se deplacer. "fleche haut"= zoom avant " fleche bas"=zoom arrière "fleche droite"=inclinaison "fleche gauche"=declinaison "o"=centrage de map (si perdu dans la map) "p"=debut d'affichage des balises names "m"= ne plus afficher les balises names. b=inverser n=deverser

Pour fermer le programme appuyez sur ECHAP -> une image au format BMP se crée automatiquement :sortie.bmp

./projet map.osm --svg permet de créer une image au format SVG. Faire make svg avant chaque usage /!\

Dans style/style.xml on a la possibilité de changer les couleurs en changant les couleurs RGBA

Si une map est trop grande, préférez l'option : ./projet map.osm --zoom car elle zoom et se déplace sur le BMP crée.

L'api overpass permet de générer puis afficher une map à l'aide des valeurs de bounds /.projet 2.36589 48.85280 2.36924 48.85410

./projet map.osm --past génère une map façon carte au trésor (les calculs sont fait de sorte à ce que la terre soit plate)

./projet map.osm --futur simule la map version futur.

Une Api permettra de donner la météo (seulement dans le terminal). Nous utilisons une Api gratuite, celle-ci ne fonctionne malheureusement que sur Paris.
